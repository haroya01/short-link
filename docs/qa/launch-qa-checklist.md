# kurl 전 표면 QA 체크리스트

> 대상: 백엔드(Spring, kurl.me) · 웹(Next.js — links 제품 + blog 제품) · iOS 블로그앱(kurl-ios) · iOS 링크앱(kurl-links-ios).
> 운용 원칙: **계층 실행** — 매 PR은 [0. 자동 게이트], 배포 직후는 [9. 프로드 스모크], 릴리스 전은 전 섹션.
> 각 항목은 "누르면 무슨 일이 일어나는가"를 실제 기기/브라우저에서 확인한다. 빌드 통과 ≠ 동작 검증.

## 2. 링크 코어 (단축 → 리다이렉트 → 통계)

- [ ] 단축: 정상 URL, 자기 자신(kurl.me) 재단축 거부, 스킴 검증
- [ ] 리다이렉트 hot path: 정상 301/302 + 클릭 기록 (p95 확인은 k6 — 사용자 직영)
- [ ] 종료 상태 HTML 게이트: 만료·한도초과·없음 — JSON 아닌 HTML, **상태코드 보존**
- [ ] 비번 링크: 오답 shake, 정답 시 잠금해제 연출, 목적지 URL data-속성 이스케이프
- [ ] maxViews 경계: 한도 정확히 도달하는 동시 요청에서 초과 통과 없음 (tx 알려진 HIGH 회귀 확인)
- [ ] QR: 생성·다운로드, zxing 버전업 후 스캔 실측
- [ ] 캠페인 빌더: 본문 다중 URL → 같은 URL 은 같은 숏링크 재사용, utm 3종 부착
- [ ] 통계: 시간대별/요일/국가/리퍼러, 봇 분리(total−human), 비공개 통계(statsPublic=false)는 게스트 404
- [ ] SSE 실시간 스트림: 연결·해제 반복에도 DB 커넥션 점유 없음 (OSIV off 회귀 — 풀 고갈 사고 이력)
- [ ] 웹훅: 등록·발송·시크릿 암호화 저장, fan-out 청크

- [ ] 표: 삽입·행열 조작·정렬 — GFM 구분선 토큰 왕복(md 내보내기→가져오기 정렬 보존), 웹·iOS 네이티브 에디터 양쪽 미러
- [ ] 이미지: 파일 첨부, 노션 붙여넣기(cross-host 302·octet-stream 포함), 혼합 붙여넣기, iOS 클립보드 이미지, half 갤러리
- [ ] 코드블록: 언어 선택, 하이라이트, 복사
- [ ] 마크다운 규율: 표준 md only — 콜아웃/형광펜 문법 없음, `:::` 등 비표준 유입 시 렌더 깨짐 없이 텍스트 처리

- [ ] 발행 다이얼로그: FeedCard replica 미리보기, 자동커버(초안만·non-dirty·해제기억), 발행 축하(sessionStorage 1회 소비)

- [ ] 리비전: 저장·비교·복원

## 4. 블로그 — 읽기 · 소셜

- [ ] 피드 4분면: 최신/인기/추천/구독함 — 전환 시 pill 활주, 각 분면 데이터 정합
- [ ] 글 상세: 제목-먼저 구조, 읽기 진행 막대, 목차(헤딩 2+), 인게이지 독(좋아요·북마크·연결) — 독이 본문 가리지 않음(접근성 크기 포함)
- [ ] 좋아요·북마크: 낙관 토글, 실패 시 롤백+에러 토스트(조용한 롤백 금지), 상/하단 버튼 동기
- [ ] 댓글: 작성·답글·좋아요·@멘션(알림 발화), 인라인 컴포저 크기(.tiptap 누수 회귀)
- [ ] 하이라이트: 문장 선택→저장, 공개 하이라이트 페인트, 내 하이라이트 목록
- [ ] 컬렉션: 담기(연결 시트 — 이미 담김 표시·why 보존), 해제, 이름 자동생성, 공개 컬렉션 열람(비로그인 permitAll — collections/1 사고 회귀), 길(path) 표식
- [ ] 팔로우: 작가·태그 — 팔로잉 목록, 팔로워 숨김 설정 반영
- [ ] 알림: A군(좋아요·댓글·팔로우)+B군(시리즈·답글·새 글 fan-out)+멘션 — 인박스 시간 묶음, 전체 읽음(성공 후에만 UI 변경)
- [ ] 검색: ?q= 결과, 트렌딩(윈도우), 인기 태그 레일 가로 스크롤
- [ ] 발견/연결 흐름: 콜드스타트(팔로우 0)에서 모든 빈 면이 다음 행동 CTA 를 가짐 (빈화면→빈화면 루프 금지)
- [ ] 리드: EMAIL_FORM 수집→목록·CSV·옵트아웃·삭제, 실패는 빈 상태로 위장 금지(ErrorState+재시도)

- 임시 prod 접근 workflow 는 **사용 후 즉시 제거** (PR 필수 규칙).
